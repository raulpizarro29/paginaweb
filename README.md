\# Mi Primera Página Web – Fundamentos de Ingeniería Informática



Este proyecto representa mi primera página web, desarrollada para la asignatura \*Fundamentos de Ingeniería Informática\*. Su creación me ha permitido comprender los principios básicos del desarrollo web y familiarizarme con la estructura.

\##  ¿Qué contiene esta página?



\- Estructura HTML, organizada en secciones claras y coherentes.

\- Hojas de estilo CSS para mejorar la presentación visual.

 Contenido informativo, como:

  -Presentación personal

  -Objetivo de la página

  -Información relacionada con el grado.

 Organización adecuada de archivos y carpetas, siguiendo buenas prácticas para facilitar el mantenimiento.

\- Preparación para despliegue, asegurando que rutas, enlaces y recursos funcionen correctamente.



\## Problemas al desarrollar la página web

Al desarrollar esta página web he tenido varios problemas como tener archivos duplicados. Tenía varias versiones de los mismos archivos dentro de docs/ y docs/public.

Tambien he tenido rutas incorrectas en el CSS ya que en algunas paginas usaba styles y en otras style. La correcta era style y las páginas que tuvieran styles aparecian en blanco.

Y otros problemas de estructura de codigo por ejemplo cuando usaba <form> o <ul>



\##  Objetivo del proyecto



El propósito principal es aplicar los conocimientos adquiridos en clase sobre HTML, CSS  creando una página funcional y bien organizada que sirva como base para futuros desarrollos más avanzados.



\##  Conclusión



La realización de esta página web ha sido mi primer acercamiento práctico al desarrollo web. Me ha permitido comprender cómo se construye y organiza un proyecto desde cero, reforzando conceptos fundamentales y sentando las bases para seguir aprendiendo y mejorando en el ámbito del desarrollo y la ingeniería informática.
